Debian Upgrade Script

This interactive shell script helps you safely upgrade your Debian system to the next stable release. It uses whiptail dialogs to guide the user through the process and ensures proper version detection and confirmation before proceeding.

Step-by-Step Process

  1. Ensure the script is run with root privileges, otherwise abort.
  2. Detect the current Debian codename (e.g., bookworm).
  3. Determine the next available Debian release (e.g., trixie).
  4. Show a confirmation dialog including current and target release names.
  5. Back up the existing /etc/apt/sources.list file.
  6. Replace all occurrences of the current codename with the target in sources.list and (if present) docker.list.
  7. Run apt update, apt full-upgrade, and apt autoremove to perform the distribution upgrade.
  8. Notify the user that the upgrade has been completed and offer a reboot.

Once finished, your system will be upgraded to the next Debian release. A reboot is strongly recommended after the upgrade to ensure all changes take effect.
