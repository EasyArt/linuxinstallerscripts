PostgreSQL & pgAdmin Installation Script

This interactive shell script installs PostgreSQL (PGDB) and/or pgAdmin on a Docker-based Linux system. The user can select which components to install via a checklist menu.

How It Works

  1. Installs whiptail if it is missing.
  2. Checks whether the Docker network produktiv exists and aborts if it does not.
  3. Deploys the selected Docker containers with automatic restart enabled.
  4. Generates secure random passwords for PostgreSQL and pgAdmin.
  5. Displays all access credentials in a final summary window.

PostgreSQL is exposed on port 5432. pgAdmin runs internally on port 80 and must be accessed via a reverse proxy.
