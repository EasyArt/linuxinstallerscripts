Beszel Installation Script

The provided script sets up a self-hosted Beszel monitoring environment.

  • Checks if whiptail is installed and installs it if missing.
  • Verifies that the script is running with root privileges.
  • Checks whether Docker is installed.
  • Verifies that the Docker network produktiv exists.
  • Creates the Docker volumes beszel_data and beszel_socket.
  • Starts the beszel container using the image henrygd/beszel:latest.
  • Sets the APP_URL environment variable to the host IP on port 8090.
  • Publishes 8090/tcp so the Beszel web interface is reachable from outside the container.
  • Uses whiptail dialogs to show error and success messages.

About Beszel

Beszel is a lightweight, self-hosted monitoring solution designed to track system and Docker host metrics. It provides a simple web interface for viewing status information and can be extended with the optional agent for monitoring additional hosts.

  • Runs as a Docker container and is easy to deploy on existing infrastructure.
  • Supports a central hub/agent setup for distributed monitoring.
  • Uses Docker volumes for persistent data storage.
  • Can be integrated into a Docker-based environment with a reverse proxy if desired.
