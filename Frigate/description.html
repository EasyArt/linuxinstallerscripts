Frigate Installation Script

The provided install_frigate.sh script installs and configures Frigate NVR as a Docker container inside an existing Docker network named produktiv.

  • Installs whiptail for interactive terminal dialogs.
  • Checks if the Docker network produktiv exists and aborts if missing.
  • Prompts the user to securely enter the FRIGATE_RTSP_PASSWORD.
  • Asks whether the system is a Raspberry Pi 4B to enable hardware acceleration.
  • Allows selection of a Google Coral accelerator:
    • none – CPU-based detection
    • USB – Coral USB Accelerator
    • PCIe – Coral PCIe Accelerator
  • Lets the user choose how Frigate media is stored:
    • Docker volume (frigate_media)
    • Bind mount to a host directory
  • Creates required Docker volumes for configuration and media storage.
  • Automatically removes an existing frigate container before redeployment.
  • Starts the Frigate container with appropriate device mappings, ports, and restart policy.

About Frigate

Frigate is an open-source Network Video Recorder (NVR) designed for real-time object detection using FFmpeg and TensorFlow. It integrates seamlessly with Home Assistant and is optimized for Docker-based deployments.

  • Supports real-time object detection on IP cameras via RTSP.
  • Hardware acceleration via VAAPI, Raspberry Pi, and Google Coral.
  • Efficient motion and object-based recording.
  • Web UI for live view, recordings, and snapshots.
  • Runs fully containerized and works well behind reverse proxies.
  • Ideal for self-hosted, privacy-focused surveillance setups.
