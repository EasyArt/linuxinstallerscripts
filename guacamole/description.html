Guacamole Docker Installation Script

This shell script installs and configures Apache Guacamole using Docker. It guides the user through the setup process and deploys the required containers.

Step-by-Step Process

  1. Prompt the user for MySQL connection details using a dialog interface.
  2. Verify Docker network produktiv exists.
  3. Create the Guacamole Docker volume if not present.
  4. Run the guacd container.
  5. Run the guacamole container with environment variables pointing to the MySQL database.
  6. Initialize the MySQL database with the required Guacamole schema and an admin user.
  7. Display a success message once installation is complete.

After the installation is finished, Guacamole is ready to use via your configured reverse proxy or hostname.
