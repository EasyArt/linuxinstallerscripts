Vaultwarden Installation Script

This interactive shell script installs Vaultwarden (a lightweight Bitwarden-compatible password manager) on a Docker-based Debian system. It uses whiptail for user feedback and checks for the necessary Docker setup.

Step-by-Step Process

  1. Verify if the Docker network produktiv exists.
  2. Create a persistent Docker volume named vaultwarden to store all data.
  3. Generate a secure 30-character admin token using openssl.
  4. Launch the Vaultwarden container with environment variable ADMIN_TOKEN, hostname vaultwarden, and attach it to the produktiv network.
  5. Display the generated admin token in a whiptail message box for the user to copy and store securely.

After completion, users will have a secure, persistent Vaultwarden instance running on Docker, with easy access to the admin panel via the generated token.
