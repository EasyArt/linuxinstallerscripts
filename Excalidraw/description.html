Excalidraw Installation Script

The provided install_excalidraw.sh script sets up a self-hosted Excalidraw environment inside an existing Docker network named produktiv.

  • Checks if whiptail is installed (and installs it if missing).
  • Verifies if the Docker network produktiv exists.
  • Removes old containers named excalidraw, excalidraw-room, and excalidraw-storage.
  • Starts three containers inside the produktiv network:
    • excalidraw – the frontend web application
    • excalidraw-room – the realtime collaboration server
    • excalidraw-storage – a community backend for scene sharing
  • Uses whiptail dialogs to show error or success messages.

About Excalidraw

Excalidraw is an open-source virtual whiteboard tool for sketching diagrams, UI mockups, and notes with a hand-drawn style. It is often described as a lightweight and privacy-friendly alternative to online whiteboards like Miro.

  • Supports drawing shapes, connectors, text, and freehand sketches.
  • Files can be exported as .excalidraw, PNG, SVG, or PDF.
  • Realtime collaboration is possible when the excalidraw-room server is running.
  • Optional excalidraw-storage backend allows users to share scenes with generated links instead of relying on the public service json.excalidraw.com.
  • Runs entirely in Docker, accessible through a reverse proxy like Nginx Proxy Manager or Traefik.
