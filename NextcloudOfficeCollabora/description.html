Collabora Online Setup Script

This script automates the setup of a Collabora Online container for integration with Nextcloud. It checks for the existing Docker network produktiv, collects your Nextcloud and Collabora FQDNs, and launches the official collabora/code image with the required configuration.

How to integrate Collabora into Nextcloud

  1. Make sure your reverse proxy points the chosen Collabora domain (e.g., collabora.example.com) to port 9980.
  2. In Nextcloud, install the Collabora Online app from the App Store.
  3. Go to Settings → Administration → Collabora Online.
  4. Enter your Collabora server URL, for example: https://collabora.example.com
  5. Save the settings and start editing documents directly in Nextcloud.
