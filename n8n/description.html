n8n Installation Script

This shell script automates the installation and configuration of n8n, an extendable workflow automation tool. The script checks for required Docker settings, prompts the user for the timezone, and deploys the n8n container in a predefined Docker network.

Step-by-Step Process

  1. Ensure whiptail is installed for interactive input.
  2. Prompt the user to input their preferred timezone (e.g., Europe/Berlin).
  3. Create a persistent Docker volume for n8n data.
  4. Deploy the n8n container with proper environment variables and volume mounts.

n8n (short for “node-by-node”) is a powerful automation platform that allows users to create custom workflows connecting different apps and services — similar to Zapier or Make, but open-source and self-hosted.

Once the script is complete, n8n is ready to use and can be accessed via the configured Docker environment.
