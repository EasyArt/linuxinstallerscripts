Meshtastic Web GUI

The Meshtastic Web GUI is a lightweight, browser-based dashboard for managing and monitoring Meshtastic devices and LoRa mesh networks. It lets you view node status, edit device settings, send and read messages, and visualize network topologyâ€”all without installing a desktop app.

Key Features

  • Live node overview: see connected nodes, battery, GPS, and radio stats.
  • Messaging: send/receive text and position updates across the mesh.
  • Config editor: update device/channel settings and region parameters.
  • Maps & topology: optional map view to inspect links and signal quality.
  • Logs & diagnostics: review events to troubleshoot connectivity.

Typical Setup

  1. Run the container: ghcr.io/meshtastic/web:latest (behind your reverse proxy if desired).
  2. Open the Web GUI in your browser and connect to your device or MQTT backend.
  3. Verify nodes appear and adjust settings as needed.

Note: Ensure your device firmware and region settings match your local regulations.
