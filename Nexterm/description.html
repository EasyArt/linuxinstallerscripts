Nexterm (All-in-One Container)

Nexterm is a modern, self-hosted web-based terminal and server management tool. It provides secure browser access to your systems, allowing you to manage servers, execute commands, and organize connections without installing a local SSH client. This script deploys Nexterm as a Docker container with persistent storage and automatic restart.

Key Features

  • Web-based terminal: access your servers directly from any browser.
  • Secure encryption: auto-generated encryption key for protecting stored data.
  • Persistent storage: configuration and data are saved in a Docker volume.
  • Auto-restart: container automatically starts after reboot or failure.
  • Simple deployment: fully automated installation via a single script.

What the Script Does

  1. Checks if the script is executed as root.
  2. Ensures whiptail is installed for dialog messages.
  3. Verifies that Docker is installed and available.
  4. Checks if port 6989 is free.
  5. Generates a secure random encryption key.
  6. Deploys the Nexterm container using the nexterm/aio:development image.
  7. Displays the access URL using the server's IP address.

Access

After installation, Nexterm will be available in your browser:
http://<your-server-ip>:6989

Note: This setup uses the development Docker image, which may be unstable. For production environments, consider switching to a stable release once available.
